=============================================================================
                          APPLICATION PURPOSE
=============================================================================

Ohm is an AI-powered IoT/Hardware Development IDE that helps makers, engineers,
and hobbyists go from idea to working prototype. It automates the complex 
process of:
- Brainstorming on User's Idea.
- Budget management [A slider which will be used as reference for parts selection]
- Component selection and compatibility checking
- Bill of Materials (BOM) generation
- Wiring diagram creation
- Code generation (Arduino C++, MicroPython)
- Hardware troubleshooting

The app eliminates the need for extensive datasheet diving and reduces the
friction between having a hardware idea and actually building it.
